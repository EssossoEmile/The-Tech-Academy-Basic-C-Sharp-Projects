# CodeFirstStudent

This is a basic **Entity Framework Code-First console application**.

- Creates a **Student database**.
- Adds **one Student** (John Doe) on first run.
- Uses EF6 Code-First approach.

## How to run

1. Open solution in Visual Studio.
2. Build solution.
3. Run the console app.
4. It will create the `SchoolDB.mdf` in your local SQL Server (LocalDB).
5. Displays all students in the database.

## Notes

- Only the initial student is added.
- Follow Code-First conventions to extend.
