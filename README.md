# My C# Projects

This repository contains multiple C# projects demonstrating skills in **ASP.NET MVC**, **Entity Framework**, **Console Applications**, and general C# programming concepts. Each project is self-contained and demonstrates different aspects of software development with C#.

---

## 1. ASP_NET_MVC_Entity_Framework_Assignment

**Description:**  
A web application built using **ASP.NET MVC** and **Entity Framework**. This project simulates a car insurance quoting system where users can create, edit, view, and delete insurance records. The application also calculates quotes based on user input.

**Key Features:**
- Full CRUD functionality for `Insuree` objects.
- Quote calculation logic based on age, car year/make/model, DUI, speeding tickets, and coverage type.
- Admin view displaying all issued quotes.
- Uses **Entity Framework** for database operations.
- MVC pattern with separate Models, Views, and Controllers.

**Technologies:** ASP.NET MVC, C#, Entity Framework, SQL Server (LocalDB)

---

## 2. CodeFirstStudent

**Description:**  
A **Code-First** console application using **Entity Framework** to create and manage a simple `Student` database. This project demonstrates creating a database from C# classes and inserting records programmatically.

**Key Features:**
- Creates a `Student` table in a SQL Server LocalDB.
- Adds one student record to the database.
- Lists all students on console output.
- Demonstrates Code-First approach in EF6.

**Technologies:** C#, .NET Framework, Entity Framework 6, SQL Server LocalDB

## 3. ConsoleAppEmployeeOperator

**Description:**  
A console application demonstrating object-oriented programming principles and operator overloading with an `Employee` class.

**Key Features:**
- Overloads comparison operators to compare employee objects.
- Demonstrates inheritance, polymorphism, and encapsulation.
- Provides practical examples of handling business logic in console apps.

**Technologies:** C#, .NET Console Application

## 4. ConsoleAppIQuittable
