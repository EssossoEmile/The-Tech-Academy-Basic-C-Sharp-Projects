# Package Express Shipping Quote Calculator

## Overview
This is a simple **C# console application** that calculates a shipping quote for a package based on its weight and dimensions. The program guides the user through inputting package details and provides an estimated shipping cost. It also enforces limits for package weight and size.

---

## Features
- Prompts the user for:
  - Package weight
  - Package width, height, and length
